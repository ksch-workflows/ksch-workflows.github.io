# KSCH Workflows Website

## Dependencies

- https://gohugo.io/
- https://themes.gohugo.io/syna/

## Run development server

```
hugo server -D
```

## Publish website

```
hugo --destination docs/
git add docs/
git commit -m "Update website"
git push
```

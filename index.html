K.S.C.H. Workflows

A custom-developed workflow management system for the Kirpal Sagar Charitable Hospital.

Video architectural prototype

Mission statement

The Kirpal Sagar Charitable Hospital is a small hospital in Punjab, India. There all people receive treatment and then pay what they can afford. It is envisioned to grow up to 500 beds and become part of a Medical College.

The purpose of this project is to support that vision. It is maintained by volunteers from the health care and software industry.

Unique Value Proposition

Simplicity

The software can be executed with minimum operational effort.

Alignment

By using custom software we can adjust it to the hospital's processes instead of vice versa.

Education

We can use it in the Kirpal Sagar Academy and universtity projects.

Charity

This is a charity project without return on investment calculations.
